Virtual Memory

Virtual memory is a memory management technique that provides an idealized abstraction of the storage resources that are actually available on a given machine.

Paging

Paging is a memory management scheme that eliminates the need for contiguous allocation of physical memory.

Page Replacement

When a page fault occurs, the operating system needs to choose which page to remove from memory to make room for the new page.
